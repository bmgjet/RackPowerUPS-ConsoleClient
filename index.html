RackPower UPS API

Each field can be accessed as raw text or as a PNG image with a transparent background.

Usage

  • Text: http://localhost:8000/FieldName
  • PNG: http://localhost:8000/FieldName.png
  • PNG with options: http://localhost:8000/size=32/color=ff0000/FieldName.png
